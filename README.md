# 📔 大人の図鑑シール コレクション

カミオジャパン「[大人の図鑑](https://kamiojapan.jp/brand-character/otonanozukan/)」シールシリーズのコレクション管理アプリです。
`index.html` をブラウザで開くだけで使えます（サーバー・インストール不要）。

## 機能

- **図鑑カタログ** — 「鉱物」「深海魚」「寿司」「レトロ喫茶」など通常版60種以上に加え、
  Premium版（W箔押し・全9種）、3D版（立体クリア仕様）も初期収録。各シールの内容説明つき。
  新作テーマは「＋追加」ボタンから登録できるほか、**「⚙ 設定 → 新テーマをさがす」で
  楽天市場の販売中商品から未収録テーマを自動発見してワンタップ追加**できます
  （画像・参考価格つきで追加されるので、新作の追いかけに便利です）。
- **カテゴリ分け** — 生き物（陸）／海・水の生き物／鳥／植物・きのこ／自然・科学／食べ物／飲み物／乗り物・建物／その他 の9カテゴリ。チップで絞り込みできます。
- **保持可否の管理** — カード右上の ✓ ボタンでワンタップで「持ってる／未入手」を切り替え。
  ダブり枚数も記録できます。
- **概要画像** — カミオジャパン**公式オンラインショップの商品画像を自動表示**します
  （週1回GitHub Actionsが公式ショップの一覧から商品名・画像URLのカタログ
  `official_catalog.json` を生成し、アプリが公式サーバーの画像を参照表示。
  画像ファイル自体はコピー・再配布しません）。さらに
  1. 楽天市場APIで実物の商品画像・参考価格・販売店を自動取得（下記）
  2. 自分で撮影した写真の登録（自動で縮小して保存）
  3. 通販サイト等の商品画像URLの貼り付け表示（Google/Yahoo!画像検索への導線つき）
  でも差し替え可能。優先順は 自分の写真 > 楽天/貼り付けURL > 公式画像 > 図鑑風カバー
  （自動生成）で、読み込めない場合はカバー画像に自動フォールバックします。
- **楽天API連携（2026年新API対応）** — [楽天ウェブサービス](https://webservice.rakuten.co.jp/)で
  アプリを登録し、「⚙ 設定」に **アプリID**（UUID形式）と **アクセスキー**（pk_〜）を入れると、
  シールごとの「楽天で商品を検索して取り込む」ボタンで候補から選んで
  画像・参考価格・商品ページリンクを登録できます。
  楽天側のアプリ設定で「アプリケーションURL（許可されたWebサイト）」に
  このアプリの公開URLのオリジン（例: `https://<ユーザー名>.github.io`）を登録してください。
  設定画面の「一括取得」を使えば、画像未設定の全テーマをまとめて取り込めます
  （APIのレート制限に合わせて1件約2秒ずつ処理。取り違えは各シールの詳細で取り直し可能）。
  ※新APIは呼び出し元サイトの検証があるため、楽天連携はWebサーバー上（GitHub Pages等）で
  開いた場合のみ動作します。file:// で開いた場合も楽天連携以外の機能はすべて使えます。
- **SNSの購買情報と連動** — シールごとに X（旧Twitter）の購入報告・入荷/在庫情報検索、
  Instagram、メルカリ／楽天ラクマ／Yahoo!フリマの出品検索へワンタップで飛べます。
  一覧画面上部には「#大人の図鑑シール」の新作・購入情報リンクも常設。
- **購入情報の連動** — 購入日・購入店・価格・メモをシールごとに記録。
  未入手のシールは Amazon／楽天／Yahoo!ショッピング／ハンズ／ロフト／公式ショップの検索リンクからすぐ探せます。
- **統計ダッシュボード** — コンプリート率、カテゴリ別の収集進捗、購入金額合計を自動集計。
- **検索・並べ替え** — 名前検索、所持状況フィルタ、図鑑順／名前順／購入日順／価格順の並べ替え。
- **バックアップ** — JSON形式で書き出し／読み込み（写真も含めて機種変更時に引っ越しできます）。
- **スマホへの新着・在庫復活通知（[ntfy](https://ntfy.sh/)連携）** — GitHub Actionsが裏で
  常時監視し、[ntfy](https://ntfy.sh/)アプリ経由でiPhone/Androidにプッシュ通知します。
  1. `zukan-watch/check_new_items.mjs`（1時間毎） — 楽天市場に新商品が並んだら通知
  2. `zukan-watch/check_stock.mjs`（下記参照） — **カミオジャパン公式ショップ・
     LOFTネットストア・PLAZAオンラインストア・
     キデイランド（Yahoo!ショッピング店舗・原宿店/通常店）**（いずれも定価販売）
     を横断で巡回し、「品切れ」だった商品が購入可能になった瞬間に、
     店名・購入リンク付きで通知

  必要なリポジトリシークレット（GitHubの Settings → Secrets and variables → Actions）:
  `RAKUTEN_APP_ID` / `RAKUTEN_ACCESS_KEY`（楽天ウォッチ用）、`NTFY_TOPIC`（両方で共通）。
  ntfyアプリで購読したトピック名を`NTFY_TOPIC`に設定してください。

  ※**ヨドバシ.com・東急ハンズ**は、robots.txtを含む全ページがクラウド
  （GitHub Actions）のIPからは常に`HTTP 403`でブロックされることを実測で
  確認したため、Actionsからの巡回対象には含めていません（取得コードは
  `zukan-watch/shops.mjs`に残置）。この2サイトは商品単位で
  **iPhoneショートカットによる自動チェック**で代替します —
  手順は [`zukan-watch/iphone_shortcut_guide.md`](../zukan-watch/iphone_shortcut_guide.md) 参照。
  他の大型サイトも同様に、将来ページ構造の変更やアクセス制限で
  一時的に取得できなくなる可能性があります（1店舗の失敗は他店舗の
  監視を止めません）。

  ### 在庫復活通知を5分間隔で確実に動かす（外部cron連携）

  GitHub純正の`schedule`実行は、5分間隔を指定しても負荷状況により
  数十分〜1時間単位に間引かれることがある**既知の制限**です（保証外の挙動）。
  確実に短い間隔で動かしたい場合は、無料の外部cronサービスから
  [`repository_dispatch`](https://docs.github.com/ja/actions/using-workflows/events-that-trigger-workflows#repository_dispatch)
  イベントを送る方式にしてください（`watch_official_stock.yml`は両対応済み）。

  1. [cron-job.org](https://cron-job.org/) 等の無料アカウントを作成
  2. GitHubで [Fine-grained personal access token](https://github.com/settings/personal-access-tokens/new) を発行
     （Repository access: このリポジトリのみ／Permissions: **Contents: Read and write**）
  3. cron-job.orgで新しいジョブを作成:
     - URL: `https://api.github.com/repos/<owner>/<repo>/dispatches`
     - Method: `POST`
     - 実行間隔: 5分ごと
     - Headers:
       - `Authorization: Bearer <発行したトークン>`
       - `Accept: application/vnd.github+json`
       - `Content-Type: application/json`
     - Body: `{"event_type":"stock-check"}`

  ※このトークンは外部サービスに預けることになるため、対象リポジトリだけ・
  必要最小限の権限に絞り、定期的に失効/再発行することをおすすめします。

## 使い方

1. `zukan-seal-collection/index.html` をダブルクリックしてブラウザで開く
2. 持っているシールのカード右上 ✓ をタップ
3. カードをタップすると詳細画面が開き、写真・購入情報を登録できます

データはブラウザの localStorage に保存されます。定期的に「書き出し」でバックアップを取ることをおすすめします。

## 初期収録テーマについて

初期データはロフト・ハンズ・公式ショップ等の販売情報をもとにした代表的なテーマのリストです。
シリーズは2017年の「鉱物」から始まり95種類以上が展開されているため、
手持ちの新テーマ・限定テーマは「＋追加」から自由に登録・編集してください。
